# Info

## Lectura de información en titulo del archivo

- Piezas: {n}pc
- Tiempo: {n}h{n}m
- Gramos: {n}g

Ejemplo: modelo_32g_5pc_2h10m.gcode
- 32 Gramos
- 5 Piezas
- 2 Horas 10 Minutos
