# Guía de instalación

## Clonar 
Puede clonar el repositorio donde más te convenga

```bash
git clone https://github.com/chepecarlos/print-tool
```

## Paquetes extras

```bash
sudo apt install python3-dev libcairo2-dev libgirepository1.0-dev pkg-config
```

## Entrara al folder del repositorio 
Debes entrar a la carpeta print-tool 

```bash
cd print-tool
```

## Submodulos 
Ejecuta lo siguiente

```bash
git submodule init
git submodule update
```

## dependencias

```bash
sudo apt install python3-tk
``` 

## instalar de paquetes

```bash
pipx install . --force
```
